CS386: Software Engineering

Course Description

This course prepares students for developing complex software systems in teams, assuming positions such as software analysts, architects, designers, developers, testers, or managers. Without adequate engineering, software projects fail due to quality, cost, or delivery problems. The course is a prerequisite for CS476 (Requirements Engineering), part of the CS capstone sequence.
